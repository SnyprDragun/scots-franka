# SCOTS for n-dimensional robotic arm reach avoid specification

## How to run files
* go to the franka folder and compile the cpp example:
  '''
  cd ~/Downloads/SCOTS_ros2_v2/SCOTS/examples/franka
  make
  '''

* run the executable
  '''
  ./franka
  '''

* run the matlab file to plot
  '''
  >> franka
  '''
